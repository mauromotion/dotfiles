# Tmux commands keybindings

**leader = <C-a>**

|## Functions             | ## Keybindings
|-------------------------|--------------------
|Move around panes        | hjkl
|Resize panes             | HJKL
|Horizontal split         | -
|Vertical split           | `|`
|Maximize pane            | m
|Move panes               | { or }
|Hot reload config        | r
|Detach from session      | d
|Attach to session        | a
|Cycle through layouts    | `space`
|New window               | c
|Rename window            | ,
|Rename session           | $
|Cycle through windows    | n
|Sessions manager         | s
|Manager with windows     | w
